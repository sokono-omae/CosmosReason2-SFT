# cosmos-reason2-app

A self-contained FastAPI web app for video action classification using CosmosReason2.

- Input: a single video file (select from the `video/` folder or upload)
- Output: live WebSocket streaming with an on-frame HUD and the latest model response
- UI language: English

## Project layout

- `fastapi_app.py`: FastAPI app (serves UI + WebSocket streaming)
- `run_fastapi_uvicorn.py`: Uvicorn launcher (disables WebSocket keepalive)
- `static/index.html`: the web UI
- `video/`: optional local videos shown in the UI dropdown
- `uv.lock`: locked dependency versions (generated by `uv`)

## Setup (uv)

Python 3.10+ is required. This project uses `uv` to create a virtual environment and manage dependencies.

Install `uv` (Linux example):

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Create the environment and install dependencies:

```bash
uv venv
uv sync
```

## Run

```bash
uv run python run_fastapi_uvicorn.py
```

Open http://localhost:8001 in your browser.


## Configuration

You can edit settings in the UI under “Hyperparameters & API settings” (collapsed by default), or via environment variables:

- `API_BASE_URL`
- `API_KEY`
- `MODEL`
- `MAX_TOKENS`
- `TEMPERATURE`
- `JPEG_QUALITY`
- `REQUEST_INTERVAL_SEC`
- `MIN_STEP_DURATION_SEC` (used for the threshold check at the end of a run)

## Action classes

The classifier uses a closed set of action classes:

- `0`: `non_action`
- `1`: `install_fan_1`
- `2`: `install_fan_2`
- `3`: `install_fan_cover_1`
- `4`: `install_fan_cover_2`
- `5`: `install_nic`

## Notes

- Videos placed under `video/` will show up in the UI's dropdown list. Uploads are stored under a temporary directory.
- The OS file picker dialog language is controlled by your browser/OS and cannot be forced from the web page.
- Graph generation and CSV export/download features are intentionally removed.
